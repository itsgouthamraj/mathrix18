  • Round 1 (Pen and Paper)

    Participants are given a list of tables and are expected to build query based on the given scenario. For the next set of questions, participants are expected to find technical terms related to DBMS by connecting the given pictures. The last set of questions determines how efficiently the participants build their query.

    Duration: 35 minutes

    Round 2 (Lab)

    Participants are given a jumbled word and set of tables from which they have to first find what the jumbled word is and query the found word from the tables. The next set of questions has tables and query for which the participants should find the output.

    Duration : 45 Minutes
